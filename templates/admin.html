TechRich Administration Library

A dedicated sandbox reference manual for Caleb to brush up on CCNA, Network architecture, and client-side helpdesk solutions. Direct click-to-draft capability included.

NETWORK ALGORITHM CCNA Routing
Resolving VLAN Leakage and Trunk Port Encapsulation Standards

Problem: Inter-VLAN traffic fails to route through a Router-on-a-Stick setup. Access terminal cannot ping secondary gateways.
Diagnostic: Check Trunk encapsulation standards. On older devices, IEEE 802.1Q override is required. Check if the Native VLAN ID matches on both switch and router virtual subinterfaces.

DYNAMIC PROTOCOLS OSPF Dynamics
Analyzing Stuck OSPF Neighbor States (DR/BDR Election Loop)

Scenario: OSPF routing adjacencies fail to transition into the **FULL** state, remaining locked inside **EXSTART** or **LOADING** sequences.
Resolution: Ensure identical Maximum Transmission Units (MTU) across physical segments. Mismatched packet sizes abort DB descriptors. Check Hello/Dead timers and unique Router IDs.

HELPDESK PROTOCOL ICT Troubleshooting
Managing Sudden DHCP Autoconfiguration Loops (169.254.x.x)

Problem: Employee workstation receives APIPA address (169.254.x.x), displaying a yellow No Internet warning flag.
Remedy: Release lease block using `ipconfig /release` followed by `ipconfig /renew`. If self-assigning persists, inspect helper-addresses on Layer 3 border boundaries pointing to DHCP servers.

DISASTER CONTROLS Network Loops
Mitigating Broadcast packet storms inside Spanning Tree Networks

Symptom: Entire workstation LAN segment freezes. Switch interfaces display extreme traffic flickers. ping response spikes up to 4,000ms.
Solution: A network loop exists on the access layer. Verify Spanning Tree election roots. Isolate interfaces. Run `spanning-tree bpduguard enable` to freeze illicit ports.
