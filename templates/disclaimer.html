Educational Disclaimer

Last updated: May 27, 2026

1. Simulation Scope (No Hostile Interaction)

The **Port Scanner Simulator**, **Password Entropy Audit**, and other interactive widgets on TechKnow Solutions are purely local client-side Javascript models. They are built for academic, educational, and pre-compliance demonstration scenarios. They do not execute external scans against third-party servers, do not inject exploit payloads, and cannot be utilized for malicious or defensive penetration tests.

2. Professional Certifications & Guidelines

Any technical instructions, checklists, command terminal inputs (e.g., `spanning-tree bpduguard enable` or `ip helper-address`), and design strategies represent best-practice Cisco networking (CCNA Enterprise) curricula. However, deploying these controls live into and modifying working topologies should only be authorized by local IT control leads or engineers.

3. Local Hosting and Code Quality

This sandbox codebase is run and deployed privately on test environments using secure sandboxed Cloud Run containers. Any live network alerts or indicators compiled by our server (such as total session records count or filtered traffic logs) are for academic performance evaluation only.
